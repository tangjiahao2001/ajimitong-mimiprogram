# 二、编程人员 注意事项

## （一）本微信小程序必须严格记录版本更替

1. 每一次提交有 `commit`的提示信息
2. 每个上线版本必需要有 `tag`标识版本号
3. 中间版本（未上线版本），文档提交后面加入tag，如 `(V1.3.4)`
4. 每一次上线版本提交需要将此版本保存 `副本`，`界面截图`。
