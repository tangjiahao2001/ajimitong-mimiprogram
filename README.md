# 二、编程人员 注意事项
## （一）本微信小程序必须严格记录版本更替
1. 每一次提交有`commit`的提示信息
2. 每一次版本提交需要补交`tag`标识版本号
3. 每一次提交需要将此版本保存副本，界面截图。
## （二）版本提交模板如下
    ## V00.00.00版本提交👇
    --------------------
    ### time
    #### 已完成
    * 完成嵌入五子棋,
    #### 未完成
    * 还有很多地方没有更名!!记得完善
